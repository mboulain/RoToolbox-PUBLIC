RoToolbox-PUBLIC is the repository I use to share the tools I made for Roblox Studio once I consider them production ready.

They are frameworks on their own, so they can be plugged in any projects.
Each tool have their own documentation for their usage.

Up to date, you can find the following tools:

- HackTracker : A tool used to simulate "hacks" in a roblox game. Perfect for making it safer. Automatically destroyed outside of studio.
- RoSpeech : A speech-to-text module, allowing you to create custom commands for interactive gameplay.
- RoJuice: A modular tool coming with multiple preset UI effects, to easily improve game feel.


I will keep adding / updating tools as I make them.

Thanks for going through them and/or using them !
